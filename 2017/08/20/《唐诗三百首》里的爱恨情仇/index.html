《唐诗三百首》里的爱恨情仇

发表于 2017-08-20 | 更新于 2017-08-20 | 分类于 人文

亲，一定记得改发布时间！

最近才正儿八经教儿子背唐诗，深感为娘的心力交瘁。捧着一本《唐诗三百首》，面对的是儿子问不完的“什么意思”，当问到自己也一知半解的时候，儿子抛来鄙视的目光“哈哈你也不会，那我走啦”，留下我独自在风中凌乱，所以我决定温故而知新。没想到拿起来随手翻了几页，就如沐春风。

唐朝的诗人不像宋朝的词人，大多数都没那么多闲情逸致写情诗，他们更喜欢写诗来表现自己洁身自爱，修身养性；或遭人排挤，满腔愤懑；又或怀才不遇，壮志未酬，大部分都是和国家社稷，官场仕途，大好河山相关的诗句，但写起儿女情长来也丝毫不逊。

燕草如碧丝， 秦桑低绿枝。
当君怀归日， 是妾断肠时。
春风不相识， 何事入罗帏。
——李白《春思》

这是李白写的思妇诗，讲女子思念远在戍边的丈夫的事。我比较喜欢最后一句，春风吹入闺房，但是女子有点不太高兴，对着春风娇嗔起来：“你又不是我丈夫，干嘛进我房间啊？”古人的忠贞可见一斑orz

上有青冥之长天，下有渌水之波澜。
天长路远魂飞苦，梦魂不到关山难。
长相思，摧心肝。
《长相思二首·其一》

“人生得意须尽欢”的李白也会被“相思”折磨到心肝摧残。不过，这里的“相思”我觉得不一定是男女之情，他可能是想念他妻子，也许还想念着唐明皇 (⊙ˍ⊙)，感觉李白对他还有所期许，他是被迫离开长安的，他的政治理想和才华都没有得到实现和施展，肯定有不甘和郁闷，“摧心肝”也自然了。李白对唐玄宗还有点“因爱生恨”的赶脚，像下面这首：

美人卷珠帘， 深坐颦蛾眉。
但见泪痕湿， 不知心恨谁。
《怨情》

明着是一首弃夫怨情的诗，其实暗喻了自己就是这“弃妇”，坐等着归来的“心上人”唐玄宗，可他就偏偏不来。

昔时横波目，今作流泪泉。
不信妾肠断，归来看取明镜前。
《长相思二首·其二》

李白真很会写思念。你看，我想你想得：以前是美目一双，现在就光顾着流泪了。你要是不信我为你断了肠，你回来时看看我疲惫憔悴的脸就晓得了。

相比李白的浪漫主义，杜甫比较现实主义，也没有太多的情爱怨恨，就一首思妻的诗：

今夜鄜州月， 闺中只独看。
遥怜小儿女， 未解忆长安。
香雾云鬟湿， 清辉玉臂寒。
何时倚虚幌， 双照泪痕干。
《月夜》

杜甫和妻子女儿分别两地，怎么能不想念呢。可杜甫想念的情诗，还是很朴实无华的。想象着妻子在月下思念自己，但小女儿却不明白母亲的相思之情，越想越凄凉，在这漫漫长夜里，她夜不能寐，云鬓被泪打湿了，没人抱，手臂也是冷的。嘤嘤嘤，没想到大文豪还是暖男一枚。

锦瑟无端五十弦， 一弦一柱思华年。
庄生晓梦迷蝴蝶， 望帝春心托杜鹃。
沧海月明珠有泪， 蓝田日暖玉生烟。
此情可待成追忆？ 只是当时已惘然。
——李商隐《锦瑟》

我惆怅，我感伤，但我不知道为什么会惆怅，为什么会感伤。

《锦瑟》是一首很朦胧的情诗，也是最难解释的一首，成为了李商隐极负盛名的代表作。说它是情诗，大抵跳不出男女之情，闺怨伤春，相思之苦，生离死别这些老梗，但都没有；说它不是情诗，但句句有无限伤感之意，让人看了都免不了替作者揪心，但还是不知道所为何故。尾联的“此情”为何情，也不得而知，也许是爱情，也许是亲情，或者是友情，抑或只是一种自伤的表达。

唐诗人甚至比宋词人还喜欢无事悲春伤秋一番，我就想赋诗一首，管他是不是有实物比照。

春心莫共花争发，一寸相思一寸灰。
《无题二首·其二》

春蚕到死丝方尽，蜡炬成灰泪始干。
《无题》

直到相思了无益，未妨惆怅是清狂。
《无题二首·其二》

李商隐的情诗都比较伤感催泪，感觉这是一个很脆弱敏感的诗人，感情到了极致，连题目都想不出来了，就直接做“无题”吧。（开个玩笑）

下面的就有一点非主旋律了。

夫人，我好想你！不过，该心动的时候还是会心动，该续弦的时候还是要续。

曾经沧海难为水，除去巫山不是云。
——元稹《离思》

我见过浩瀚大海里的水，其他的水，那还叫水啊；我见过巫山上的神女之云，其他的云真特么难看了！元稹把他亡妻比作世间唯一的珍宝，没人比得上了，其他女人都不入其法眼。

谢公最小偏怜女，自嫁黔娄百事乖。
顾我无衣搜荩箧，泥他沽酒拔金钗。
野蔬充膳甘长藿，落叶添薪仰古槐。
今日俸钱过十万，与君营奠复营斋。
——元稹《遣悲怀三首》 其一

妻子死了，我好难过。她本是官宦千金，委身下嫁于我，和我一起过苦日子，还照顾得我好好的，啥抱怨也没有。在世的时候对我可好了：看到我没衣服穿了就翻箱倒柜地去找，我想喝酒了就把头钗拿去当了换钱，只能吃野蔬菜充饥，她却吃得津津有味，没柴火了，就用老槐树的落叶烧柴（古人都很喜欢有点夸张地说自己过得好凄惨ヽ(；´Д｀)ﾉ）。现在我赚再多的钱又怎么样，还是不能和她共享富贵，心塞(〒︿〒)。只能多祭奠，再请法师超度亡灵以表自己的愧疚和悼念之情。

元稹还有一句最广为流传的是《其二》：

诚知此恨人人有，贫贱夫妻百事哀。

想起以前我们共患难同贫贱的时候，我就哀伤不已啊。

但是现在“贫贱夫妻百事哀”更多是指没有经济基础的夫妻日子过得吃土的意思。古人还是比我们超脱多了，有爱的夫妻就算吃土也甘之如饴。看得出来，元稹对亡妻真是情深义重。

但素，没过多久，元稹又看上别的女人，过几年又续弦娶了另外一个女人。

文学作品是一回事，生活是另一回事，但我相信那句“曾经沧海难为水”彼时应该还是真心的。

诗人多情又有才，是很容易招蜂引蝶的。

十年一觉扬州梦，赢得青楼薄幸名。
——杜牧《遣怀》

So，这还是很光荣的事情吗？

杜牧说我是风流诗人No.1，估计没人敢吭声。在扬州做官时期和青楼女子来往颇深，诗酒风流，放荡形骸。没有佳人作伴的才子，总归是不圆满的，这都可以理解。

娉娉袅袅十三余，豆蔻梢头二月初。

十三岁的歌姬，又美又瘦，豆蔻年华，叫我怎得不喜欢。

春风十里扬州路，卷上珠帘总不如。

唐代的扬州当时经济文化极度繁荣，十里长街，车水马龙，但这春风十里扬州路上不计其数的珠帘美人，在杜牧心里都比不上眼前的这位呀。

可是，天下无不散之筵席，杜牧要离开了。

多情却似总无情， 唯觉樽前笑不成。

不忍分别，又不得不分别。杜牧太喜欢她了，以至于他觉得用什么辞藻都表达不出这样的感情，越是多情，就越显得无情，离开的时候，四目相对，又无言以对。

蜡烛有心还惜别，替人垂泪到天明。

在老杜眼里，蜡烛都在替自己和美人的分别流泪啊。到了天明，他俩还是舍不得离开。

杜牧对青楼女子有一种“英雄惜英雄”的感情，他还有一首专门吊唁绿珠的：

繁华事散逐香尘，流水无情草自春。
日暮东风怨啼鸟，落花犹似坠楼人。
——《金谷园》

当然杜牧也不全是赞扬她们的美好。

烟笼寒水月笼沙， 夜泊秦淮近酒家。
商女不知亡国恨， 隔江犹唱后庭花。
——《泊秦淮》

其实歌女们也是无奈，客人点什么就得唱什么，不是我们“不知亡国恨”啊。

杜牧“性疏野放荡”不是一个人，还有个叫韩绰的陪着一起。我是一直相信男人之间的友情坚若磐石。

青山隐隐水迢迢， 秋尽江南草木凋。
二十四桥明月夜， 玉人何处教吹箫。
——《寄扬州韩绰判官》

这里的“玉人”就是韩绰啦。杜牧和韩绰相隔两地，好久没见，甚是想念，但还不忘调侃一番：我不在的时候，你这小子每天晚上在哪里教歌妓唱歌吹箫呀？

杜牧这个人不是一般的爱美女，还有很多诗是和他的处处留情有关，但没有收录进《唐诗三百首》里。我觉得风流才子多情是没问题的，但是最好能“我挥一挥衣袖，不带走一片云彩”，别动不动就体现在自己的作品里，搞得自己比婉约派还“婉约”，《唐诗三百首》里就属他的情诗最多最香艳了。

回眸一笑百媚生，六宫粉黛无颜色。
…
春宵苦短日高起，从此君王不早朝。
…
后宫佳丽三千人，三千宠爱在一身。
…
上穷碧落下黄泉，两处茫茫皆不见。
…
在天愿作比翼鸟，在地愿为连理枝。
天长地久有时尽，此恨绵绵无绝期。
——白居易《长恨歌》

压轴出场的白老写别人的爱情故事，就跟亲身经历过一样，荡气回肠，惊天动地，自己都要被自己感动死了。
